To use the program, simply execute the jar.
You will be prompted with 4 options:
Add
Search
Display
Quit

Select any of theese options by simply typing them into the prompt
You will then be asked to specificy a book to add or a book to search,
depending on your choice.

Once you are satisfied, The command "Quit" will exit the program

Any incorrect or unknown commands will prompt you to enter them again.
